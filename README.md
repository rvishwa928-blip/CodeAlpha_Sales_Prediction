# Sales Prediction using Python

## Description
This project predicts sales based on advertising spend,
marketing platform, and target customer segment using
Machine Learning techniques.

## Dataset
- Automatically generated sales dataset
- Includes advertising spend, platform, and target segment

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Model Used
- Linear Regression

## Workflow
1. Dataset creation and loading
2. Data cleaning and preprocessing
3. Encoding categorical variables
4. Train-test split
5. Model training
6. Prediction and evaluation
7. Visualization of results
8. Marketing insights extraction

## Key Insights
- Advertising spend has the highest impact on sales
- Platform and target segment also influence sales outcomes
